2. 概要
-------
   USB インターフェイスのカスタマーディスプレイを、仮想シリアル  ポートとして
   認識させるドライバーです。アプリケーションからは、仮想シリアルポートを指定
   することで、カスタマーディスプレイを制御できます。

   ファイル構成
     DM-D(USB)_VCPdriver_Verxxx            パッケージのフォルダ
        |
        |---Driver                         DM-D Virtual COM Port Driver のドライバーが格納されているフォルダ
        |                                  インストールする際、このフォルダを指定する
        |
        |---Manual                         マニュアルの格納フォルダ
        |                                    DM-D_VCOMDrv_um_en_revA.pdf
        |                                    DM-D_VCOMDrv_um_jp_revA.pdf
        |
       README_jp.txt                        README ファイル(日本語)
       README_en.txt                        README ファイル(英語)

   インストール方法
     DM-D_VCOMDrv_um_jp_revA.pdfをご参照ください。
